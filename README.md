# Customer Churn Analysis

## Project Overview
This project analyzes customer churn data to understand why customers leave a company and builds predictive models to identify customers who are likely to churn.

## Objectives
* Explore customer behavior and churn patterns.
* Handle missing values and prepare data for analysis.
* Visualize important factors affecting churn.
* Build machine learning models for churn prediction.
* Improve prediction performance using SMOTE.

## Tools & Technologies
* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* SMOTE (Imbalanced-learn)

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Missing Value Treatment
4. Exploratory Data Analysis (EDA)
5. Feature Encoding
6. Data Splitting
7. Model Training
   * Logistic Regression
   * Random Forest Classifier
8. Model Evaluation
9. Handling Class Imbalance with SMOTE

## Key Insights
* Contract type and internet services influence customer churn.
* Data visualization helps identify churn trends.
* Machine learning models can predict customers at risk of leaving.

## Results
Models are evaluated using:

* Accuracy Score
* Recall Score
* ROC-AUC Score
